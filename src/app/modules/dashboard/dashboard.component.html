Smartcampus bietet Zugriff auf Daten zu Gebäuden am Campus des Karlsruher Institut für Technologie. Diese Daten umfassen unter anderem Barrierefreiheitsdaten, Einzelheiten zu Räumen von Interesse sowie Probleme bezüglich einzelner Elemente in einem Gebäude oder auch bezüglich des gesamten Gebäudes. Probleme können von jedem angemeldeten Benutzer gemeldet werden. Diese werden dann auf Richtigkeit und Dringlichkeit überprüft und dementsrpechend behoben. Den Fortschritt eines gemeldeten Problems kann man dabei als angemeldeter Benutzer hier auf der Website überprüfen. Zu den bereitgestellten Services gehören:

  • Betrachten, Hinzufügen, Entfernen und Bearbeiten von Gebäuden / Räumen / Komponenten
    Als Admin können in der Datenbank vorhandene Objekte über den Bearbeiten-Button im User Interface hinzugefügt, bearbeitet und gelöscht werden. Alle zu Gebäuden zugehörige Daten können über das System gesucht (zum Beispiel über den Namen eines Gebäudes) und eingesehen werden.

  • Favorisieren von Gebäuden, Räumen oder Komponenten
    Gebäude / Räume / Komponenten können in ihrer jeweiligen Ansicht über den Favorisieren-Button zur Favoritenliste des Benutzers hinzugefügt werden.

  • Betrachten und Verwalten der Favoriten
    Als Benutzer kann man seine bereits favorisierten Gebäude / Räume / Komponenten einsehen und anpassen.

  • Melden von Problem bezüglich Gebäuden, Räumen oder Komponenten
    Probleme können von jedem angemeldeten Benutzer gemeldet werden. Diese werden dann auf Richtigkeit und Dringlichkeit überprüft und dementsrpechend behoben.

  • Betrachten und Verwalten von eigenenallen bereits gemeldeten Problemen
    Als Admin können alle von Benutzern gemeldeten Probleme eingesehen und bearbeitet werden. Dabei können sowohl Titel und Beschreibung des Problems, als auch sein Status verändert werden. Als Benutzer kann man seine gemeldeten Probleme einsehen. Ein Problem kann den Status "OPEN", "ACCEPTED", "IN_PROGRESS", "DECLINED" oder "CLOSED" haben.
